13 Setups a data test class that is ready for execution.

19 {'not_null': {'columns': ['FIRST_SCAN_ITEM_ID', 'BOOK_TO_SHOW_SCORE']}} 

20

22 sdc_data_validation.validation_test_enums.ValidationTests. This will provide all the information related to 

23 the data test library and class. The value validation_test_info_ will be unpacked and passed into the 

24 data test class being called. In the example above, "columns" would be passed in as an argument to the 

25 ColumnNotNull class constructor. 

26 

27 :param run_against_: String. Target to run data tests against. Options: 

28 - "dataframe": Test will be run on an SDCDataframe. "sdc_dataframe_" must be passed in. 

29 - "database": Test will be run on a database table. "database_info_" must be passed in. 

30 :param sdc_dataframe_: SDCDataframe. Dataframe to run test against (if selected). 

31 :param data_schema_: Dict. Full SDC data schema. 

32 :param endpoint_schema_: Dict. SDC endpoint data schema. 

33 :param database_info_: Dict. Database client and information related to how to access 

34 the table to validate. 

35 :param test_number_: Int. If used, the test number for keep track of the order of test runs. 

36 

37 Example database_info_: 

38 

39 { 

40 'database_client': SnowflakeDatabase 

41 'database_type': 'snowflake', 

42 'database_name': 'MARKETING', 

43 'schema_name': 'IMPACT', 

44 'table_name': 'ACTIONS_BY_DAY' 

45 } 

46 

47 :return: Class instance of a data test. 

48 """
